This is Perl module B<bayatest>.


=head1 INSTALLATION

bayatest installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan bayatest

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install


=head1 DOCUMENTATION

bayatest documentation is available as in POD. So you can do:

    % perldoc bayatest
